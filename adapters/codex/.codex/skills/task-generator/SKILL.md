# Task Generator

Create a task using `templates/current-task.md`.

Include:

- Goal
- User value
- Scope
- Out of scope
- Acceptance criteria
- Likely files or areas
- Risks
- Assumptions

Rules:

- Do not implement.
- Keep task focused.
- Make acceptance criteria testable.
- Call out missing decisions.
